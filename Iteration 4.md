# Iteration 4

The goal of this iteration is to add a widget to `./public/index.html` that allows the user to set the current between 0.00 and 2.00 amperes in 0.1A steps.

The widget should be displayed in the middle of the page and should be styled to be easily visible and easy to use.

## Context

You will add a proof of concept functionality to the application's GUI to allow the user to set the current between 0.00 and 2.00 amperes in 0.1A steps.

## Task

Your task is to research, analyze and understand how to add current selection by the user using the GUI.

1. Add to `./public/index.html` a widget that allows the user to set the current between 0.00 and 2.00 amperes in 0.1A steps.
2. Modify `SerialControllerApp.java` so the changes of the current setting will be reflected replacing the currently static value of 2.00 amperes.
3. You will use simple HTML, CSS and JavaScript to implement the widget.
